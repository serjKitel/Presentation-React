What is React?

  • React is a JavaScript library which is used for creating user interfaces.
  • React was intended for the web, for creating websites.
  • React is considered an ideal tool for building scalable web applications.
  • React is relatively easy to learn, with a clear and concise syntax.

Virtual DOM

Virtual DOM

  • The virtual DOM is a lightweight copy of the real DOM. And a distinctive feature of React is that this library works with the virtual DOM, and not with the real one.
  • If the application needs to know information about the state of elements, it accesses the Virtual DOM.
  • If elements of a web page need to be modified, the changes are first made to the virtual DOM. Then the new state of the virtual DOM is compared with the current state. If these states differ, React finds the minimum number of manipulations required to update the real DOM to the new state and performs them.

What are React components?

React provides us with two ways of creating a component:

  • Class Component
  • Functional Component

The component-based approach includes two key terms:

  • Composition
  • Decomposition
